
- Backend частина для проекту стримінгової платформи.
- Під час роботи з ментором відбулося ознайомлення та робота з:
1) P2P(протокол)
2) WebTorrent
3) TypeScript
4) Node.js(Express)
5) MongoDB
6) Робта з зовнішнім API(IMDB)
7) Web scraping
8) Docker

Функціонал дозволяе:
- знаходження та перегляд матеріалу в лайв-режимі(поки що не всі формати відео)
- скрапінг даних про фільми з IMDB
